The Inner Workings of the Internet

    As stated above, when you click on a link, a whole process occurs in a split second. Depending on your internet speed of course.

    • Once clicked, a secured request is sent to the website
    • The request first goes to your ISP, which attempts to match the websites name with the IP address
    • With the IP address located, the request finds the fastest way possible through many routers
    • Once there, the server sends a secured response, which quickly makes its way back to you, thus finally completing the proccess
